# 📐 Calculadora de Equação do 2º Grau (HTML5)

Este projeto é uma calculadora interativa de equações do 2º grau desenvolvida em HTML5, JavaScript e Chart.js.

Ela resolve automaticamente a equação no formato:
ax² + bx + c = 0


e exibe:
- Passo a passo do cálculo
- Valor do delta (Δ)
- Aplicação da fórmula de Bhaskara
- Raízes reais (x1 e x2)
- Gráfico da função quadrática

---

## 🚀 Funcionalidades

✔ Entrada de valores (a, b, c)  
✔ Cálculo automático do delta  
✔ Resolução detalhada (Bhaskara)  
✔ Identificação de raízes reais ou inexistentes  
✔ Gráfico da parábola  
✔ Interface simples e leve  

---

## 📊 Tecnologias usadas

- HTML5
- CSS3
- JavaScript
- [Chart.js](https://www.chartjs.org/) (para o gráfico)
